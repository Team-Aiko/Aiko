# 1. Project: Aiko (프로젝트 소개)

-   `Goal`: 기존의 ERP는 각 회사별로 별도의 ERP를 구축해 사용해왔고 이를 도입하는데 많은 비용을 요구한다. Aiko는 기업의 ERP 제공플랫폼으로서 별도의 도입비용을 절감하고, ERP의 한계였던 기업 폐쇄적 시스템을 극복해 기업간 커뮤니케이션을 구축가능하게 하는데 목적이 있다.
